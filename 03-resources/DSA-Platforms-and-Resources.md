# 📚 DSA Platforms & Resources — Complete Reference

> All platforms are free unless marked otherwise. Bookmarked and categorised by purpose.

---

## 🥇 Tier 1 — Interview Focused (Start Here)

### 1. roadmap.sh — DSA Interactive Roadmap
🔗 **https://roadmap.sh/datastructures-and-algorithms**

- **Best visual interactive roadmap available** — 6th most starred project on GitHub
- Every topic is a clickable node — mark as **Done / In Progress / Skip**
- Covers the full spectrum: Arrays → Linked Lists → Trees → Graphs → DP → Tries → Segment Trees → Union-Find → all problem-solving paradigms
- Also has a downloadable PDF version
- **Use this to:** See all DSA topics at a glance, track your progress, and identify gaps

> 💡 **Tip**: Open this alongside our repo — cross-reference each node with the corresponding deep-dive file here.

---

### 2. AlgoMap.io — Topic-by-Topic Roadmap with LeetCode Links
🔗 **https://algomap.io/roadmap**

- Free structured roadmap directly mapped to **LeetCode problems**
- Topics: Arrays, Strings, HashMaps, Two Pointers, Stacks & Queues, Linked Lists, Binary Search, Sliding Window, Trees, Heaps & Priority Queues, Recursion, Backtracking, Graphs, DP
- Each topic shows **which LeetCode problems to solve and in what order**
- Progress tracking built in
- **Use this to:** Practice problems in the right order after studying theory

---

### 3. GeeksForGeeks — DSA Roadmap + SDE Sheet
🔗 **https://www.geeksforgeeks.org/dsa/complete-roadmap-to-learn-dsa-from-scratch/**
🔗 **SDE Sheet: https://www.geeksforgeeks.org/sde-sheet-a-complete-guide-for-sde-preparation/**

- Comprehensive written roadmap with explanations per topic
- **SDE Sheet** = curated list of must-solve problems for Software Development Engineering interviews
- Covers: Time/Space complexity → Core DS → Algorithms → Advanced topics
- Each topic links to articles, problems, and video explanations
- **Use this to:** Deep-read any topic + solve the SDE Sheet problems systematically

---

### 4. LeetCode — Practice Platform
🔗 **https://leetcode.com**

- The gold standard for interview practice
- **Blind 75**: https://leetcode.com/discuss/general-discussion/460599/blind-75-leetcode-questions
- **NeetCode 150**: https://neetcode.io/practice (curated 150 problems by pattern)
- Filter by: Topic → Difficulty → Company
- **Use this to:** Solve problems after studying each topic

## 🥈 Tier 2 — Deep Dive & Competitive Programming (0 to ∞)

### 5. GitHub — Competitive Programming & DSA ROADMAP
🔗 **https://github.com/The-CODE-Plus-Plus-Community/Competitive-Programming-and-DSA/blob/main/ROADMAP.md**

- Goes **far beyond interview prep** — full competitive programming depth
- Covers topics not found elsewhere:
  - Number Theory (all of it)
  - Game Theory (Sprague-Grundy Theorem)
  - Aho-Corasick multi-pattern matching
  - Suffix Arrays & Suffix Automaton
  - Floyd Cycle Detection
  - Combinatorics & Probability Theory
  - BigInteger arithmetic
- **Use this to:** Go from interview-ready to competitive programming level

---

### 6. VisuAlgo — Algorithm Visualizations
🔗 **https://visualgo.net**

- Developed at National University of Singapore (NUS)
- **Animated step-by-step visualizations** of 23+ algorithms
- Covers: Sorting, Linked List, Hash Table, BST, Heap, Graph Traversal, MST, SSSP (Dijkstra, Bellman-Ford), Union-Find, Trie, Suffix Array, and more
- Watch each algorithm execute on your own custom input
- **Use this to:** Understand HOW an algorithm works visually before coding it

> 💡 **Best use**: Before starting a new topic (e.g., Dijkstra), spend 10 minutes on VisuAlgo watching it animate — it makes the code 10x easier to understand.

---

### 7. CP-Algorithms (cp-algorithms.com)
🔗 **https://cp-algorithms.com**

- Open-source reference site — the **Wikipedia of competitive programming algorithms**
- Covers everything from basic to extremely advanced:
  - All graph algorithms with mathematical proofs
  - Number theory, combinatorics, geometry
  - String algorithms (KMP, Z-function, Suffix Array, Aho-Corasick, Manacher)
  - Advanced DP (Divide & Conquer DP, Knuth's optimization, CHT)
  - Data structures (Segment Tree, Fenwick, Sparse Table, DSU)
- Each article includes: concept explanation + proof + implementation (C++)
- **Use this to:** Get the deepest possible understanding of any algorithm

## 🥉 Tier 3 — Practice Platforms with Built-in Roadmap

### 8. CodeChef DSA Roadmap
🔗 **https://www.codechef.com/roadmap/data-structures-and-algorithms**

- Structured roadmap from fundamentals to advanced
- 450+ curated practice problems
- Goes from Arrays & Linked Lists → DP & Graph Algorithms
- Good for structured daily practice
- **Use this to:** Daily problem solving with a built-in progression system

---

### 9. HackerRank — Interview Preparation Kit
🔗 **https://www.hackerrank.com/interview/interview-preparation-kit**

- Curated interview prep kit with problems grouped by topic
- Good for: Arrays, Strings, Sorting, Searching, Graph Theory, DP, Stacks & Queues
- Companies use HackerRank for screening tests — good to be familiar with the platform
- **Use this to:** Practice in a timed, test-like environment

---

### 10. Codeforces — Competitive Programming
🔗 **https://codeforces.com**

- The top competitive programming platform
- Problems rated by difficulty (800 = beginner → 3500 = world-class)
- **Filter by topic**: https://codeforces.com/problemset (use tags)
- Regular contests (Div 1, Div 2, Div 3, Div 4)
- **Use this to:** Build problem-solving speed and depth after mastering theory

## 🎯 Bonus Resources

### NeetCode — YouTube + Practice
🔗 **https://neetcode.io**
🔗 **YouTube: https://www.youtube.com/@NeetCode**

- Best YouTube channel for DSA interview prep
- Every video explains a LeetCode problem with: intuition → approach → code → complexity
- **NeetCode 150** = curated 150 problems covering all patterns
- **Use this to:** Watch video explanations when stuck on a problem

---

### USFCA Data Structure Visualizations
🔗 **https://www.cs.usfca.edu/~galles/visualization/Algorithms.html**

- University of San Francisco's interactive algorithm visualizations
- Written in JavaScript + HTML5 Canvas — works in any browser
- Covers: AVL Trees, B-Trees, Hash Tables, Heap, Sorting algorithms
- **Use this to:** Visualize lesser-covered data structures like AVL, B-Trees

---

### The Algorithms — Open Source Algorithm Implementations
🔗 **https://github.com/TheAlgorithms/Java**

- Open-source repository with implementations of every algorithm in Java
- Organized by category: Sorts, Searches, Graphs, Dynamic Programming, etc.
- **Use this to:** Reference clean implementations when studying a new algorithm
